# Dra. Helena Costa — Site de Estética Premium

Site institucional profissional para clínica de estética, desenvolvido com React + TypeScript + Tailwind CSS + Vite.

## 🚀 Tecnologias

- **React 18** — Biblioteca UI
- **TypeScript** — Tipagem estática
- **Tailwind CSS** — Estilização utilitária
- **Vite** — Build tool rápido
- **Framer Motion** — Animações
- **Lucide React** — Ícones

## 📁 Estrutura

```
src/
  components/
    Navbar.tsx         — Navegação fixa com scroll effect
    Footer.tsx         — Rodapé completo
    WhatsAppButton.tsx — Botão flutuante WhatsApp
  sections/
    Hero.tsx           — Banner principal
    About.tsx          — Sobre a profissional
    Services.tsx       — Tratamentos e preços
    Gallery.tsx        — Galeria com lightbox
    Testimonials.tsx   — Depoimentos
    Contact.tsx        — Formulário + info de contato
  App.tsx              — Componente principal
  main.tsx             — Entry point
  index.css            — Estilos globais
```

## 🎨 Paleta de Cores

- **Gold** — Dourado elegante (#d4922a)
- **Cream** — Fundo creme (#faf8f5)
- **Dark** — Texto escuro (#1a1a1a)

## 📱 Seções

1. **Hero** — Banner com parallax, CTA duplo
2. **Sobre** — Bio, estatísticas, valores
3. **Tratamentos** — 8 serviços com preços
4. **Galeria** — Grid com filtro e lightbox
5. **Depoimentos** — 4 depoimentos de clientes
6. **Contato** — Formulário + info + WhatsApp

## 🚀 Deploy na Vercel

1. Suba o código no GitHub
2. Conecte o repositório na Vercel
3. Build command: `npm run build`
4. Output directory: `dist`

## 📝 Personalização

Substitua as imagens em `/public/` pelas fotos reais da cliente.
Atualize os textos genéricos (nomes, endereço, telefone) nos componentes.
